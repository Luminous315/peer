# Personal Resume Website

这是我的个人简历网站，用于展示个人背景、实习经历、技能能力和联系方式。

## Preview

网站发布地址：

https://luminous315.github.io/peer/

## Sections

- Opening Bookmarks Section
- About Me
- Skills
- Work Experience
- Contact Me

## Features

- 互动书签式开场页面
- 中英文切换功能
- 滚动联动标题过渡效果
- 文件夹式 Skills 展示
- Work Experience 悬停切换展示
- Contact 信息点击复制
- 响应式布局，适配桌面端和移动端

## Tech Stack

- HTML
- CSS
- JavaScript
- Tailwind CSS

## Notes

本网站仅用于个人求职展示。  
未公开的毕业课题数据、实验结果、论文内容及实习内部资料不会在网站中展示。
